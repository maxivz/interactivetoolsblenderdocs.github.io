#Uv Tools
##Rotate 90+
Rotates the UV selection 90 degrees.

##Rotate 90-
Rotates the UV selection -90 degrees.

##Seams From Sharps
Adds seams from Sharp Edges

##Seams From Islands
Adds seams from Uv Islands in the current uv channel.

##Uvs From Sharps
Applies seams from sharp edges and unwraps everything.
